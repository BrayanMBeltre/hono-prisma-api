# HONO and Prisma api

This is a Bunjs project using Hono to manage users with prisma.

## Installation

First, clone the repository:

```bash
git clone <repository-url>
cd <repository-name>
```

Then install dependencies and check that it works

```bash
bun install
bun dev
```

## Build

```bash
bun run build
bun run start
```
